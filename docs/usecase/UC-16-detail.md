- 対象事業: チワワファミリーの家庭内ペット健康管理（家族共有型 Web アプリ）
- 入力骨格: `docs/catalog/use-case-skeleton.md`（UC-16 行）
- 入力事業要件: `docs/business-requirement.md`（§5 Key Issue I5「ルナの低血糖・ワクチン管理」／§10.4「ルナ（2歳）の低血糖予防に関する家庭内対応プロトコル（給餌頻度・緊急時手順）」）
