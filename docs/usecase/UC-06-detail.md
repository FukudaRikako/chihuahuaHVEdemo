# UC-06 詳細 — 家族メンバーが月次予防薬（フィラリア/ノミダニ）投与を記録する

- 親骨格: `docs/catalog/use-case-skeleton.md` UC-06
- 入力: `docs/business-requirement.md`（As-Is §4 / Key Issues §5 / To-Be §6 / Strategic Recommendations §9）
- スコープ: 推奨戦略「オプション B（家族共有型 Web アプリ MVP→拡張）」。診断行為・家族外共有・紙運用継続は対象外。

## ユースケース定義

- **ID**: UC-06
- **名称**: 家族メンバーが月次予防薬（フィラリア/ノミダニ）投与を記録する
- **業務エリア or ジャーニー段階**: 月次予防業務（フィラリア／ノミダニ予防薬投与の実施と記録）
- **目的**: 4 匹それぞれの月次予防薬投与の「やった／やっていない」を家族全員に同期し、投与漏れと二重投与を防ぐ。
- **提言/論点ID（REC-xx）**: REC-01（家族共有型 Web アプリによる記録・リマインド・通院前サマリ、`docs/business-requirement.md` §9 推奨戦略「オプション B」）
- **一次アクター**: 家族メンバー（投薬実施者）
- **二次アクター/外部システム**: システム（スケジューラ：UC-14 連携）、他家族メンバー（同期受信側）
- **トリガー**: 家族メンバーが月次予防薬を投与した直後、または UC-14 リマインド通知から本記録画面に遷移したとき。
- **前提条件**:
  - UC-01 でログイン済み、かつ UC-02 で家族グループに所属している。
  - UC-03 で対象犬（4 匹のいずれか）のプロフィールが登録されている。
  - 対象犬に対し月次予防薬の種別（フィラリア／ノミダニ）と投与周期（月次）が設定されている。
- **基本フロー要約**:
  1. 家族メンバーが「今日のタスク」または UC-14 通知から月次予防薬タスクを開く。
  2. 対象犬と予防薬種別（フィラリア／ノミダニ）を確認し、投与日時（既定=現在時刻）と実施者を入力する。
  3. 「投与済み」として記録を保存する。
  4. システムは記録を家族グループへ同期し、当該タスクを完了状態に更新する。
  5. システムは次回投与予定日（+1 か月）を算出し、UC-14 リマインド対象としてスケジュールする。
- **主要代替/例外**:
  - 同月内に他家族が既に「投与済み」を記録 → 二重投与確認ダイアログを表示し、保存を中止または上書き理由付きで継続。
  - 投与をスキップした（嘔吐・体調不良等） → 「スキップ」を理由メモ付きで記録し、次回投与予定日は据え置き／要再投与フラグを立てる。
  - オフラインで保存不可 → ローカル下書きとして保持し、オンライン復帰時に同期。
- **事後条件**:
  - 対象犬・対象月の月次予防薬実施ログが家族グループ全員に同期されている。
  - 次回投与予定日が更新され、UC-14 がリマインド対象として認識している。
  - UC-20 KPI ダッシュボードの「予防薬遵守率」算出元データが更新されている。
- **主要データI/O（データオブジェクト名のみ）**:
  - 入力: 犬プロフィール、月次予防薬タスク、投与種別、投与日時、実施者、（任意）理由メモ。
  - 出力: 月次予防薬実施ログ、次回投与予定、家族向け同期イベント。
- **データSoR/オーナー部門**: 本アプリ（家族共有型 Web アプリ）が SoR。オーナーは主家族メンバー（家族グループ管理者）。
- **依存システム**: 認証基盤（UC-01）、家族グループ管理（UC-02/UC-19）、スケジューラ（UC-14）、KPI 集計（UC-20）、通院前サマリ生成（UC-12）。
- **In範囲**: 月次予防薬（フィラリア／ノミダニ）の投与実績記録、次回予定の自動更新、家族同期、スキップ理由の記録。
- **Out範囲**: 予防薬の医学的妥当性判定、用量計算、購入・在庫管理、外部 EC／薬局連携、獣医師アカウントによる直接記録。
- **Out理由（短文）**: 診断行為を含めない方針（`docs/business-requirement.md` §9）、家族外への直接アカウント発行は対象外（骨格 Out of Scope）、EC／在庫／IoT 連携は戦略スコープ外。
- **KPI紐づけ**: 予防薬遵守率（`docs/business-requirement.md` §9 主要 KPI）、投薬・予防薬の記録漏れ 0 件運用化。KPI/OKR ID は `docs/recommended-kpi-okr.md` 未生成のため TBD。
- **優先度**: P0（I1 投薬・予防薬漏れ／I5 ルナの予防管理を直接解消、MVP 短期スコープに明記）。

  - TBD: スキップ時の再投与許容期間と要再投与フラグ運用／影響: 遵守率 KPI 算出定義／埋め方: かかりつけ動物病院への確認可否を家族代表に問合せ／宛先: 主家族メンバー＋（任意）獣医師／B。
