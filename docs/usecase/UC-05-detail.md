# UC-05 詳細: 家族メンバーが投薬実施を記録しチェック完了する

- 対象事業: チワワファミリーの家庭内ペット健康管理（家族共有型 Web アプリ）
- 入力骨格: `docs/catalog/use-case-skeleton.md` の UC-05
- 関連 Key Issue: I1（投薬漏れ）, I6（家族間タスク重複）
- 関連 Strategic Recommendation: REC-01（記録・リマインド・通院前サマリに機能を絞った家族共有型 Web アプリ／オプション B MVP）
- スコープ前提: 戦略オプション B が提供するソフトウェアシナリオに限定。診断・家族外共有・紙運用継続は対象外。

## セルフチェック

- スコープ整合: 戦略オプション B（推奨）のソフトウェア提供範囲内であり、診断・家族外共有を含まない。
- 粒度: 一次アクターのゴール「投薬を実施し家族で共有完了状態にする」単位で、画面名／単一処理名で終わっていない。
- 関連 UC 連携: UC-01（ログイン）／UC-03（犬プロフィール）／UC-04（今日のタスク）／UC-10（タイムライン）／UC-14（リマインド通知）／UC-20（KPI）と明示的に接続。
- 課題カバレッジ: I1（投薬漏れ）、I6（家族間タスク重複）の双方を解消する設計。
- 必須列: 全項目記入済み。不明は `TBD` ＋理由／影響／埋め方／宛先／B|NB を明記。
