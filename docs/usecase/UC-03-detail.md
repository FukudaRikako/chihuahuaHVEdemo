# UC-03 詳細 — 家族メンバーが犬プロフィールを登録・編集する

- 対象事業: チワワファミリーの家庭内ペット健康管理（家族共有型 Web アプリ）
- 関連戦略: REC-01（推奨オプション B：専用 Web アプリ MVP→拡張）の MVP スコープ
- 関連課題: I7（写真・症状分散）、I2（通院前情報整理）、I4（シニア犬モニタ）の前提となる「犬軸統合台帳」の起点
- 入力骨格: `docs/catalog/use-case-skeleton.md` UC-03
- 上位事業要件: `docs/business-requirement.md` §6 To-Be / §9 Strategic Recommendations 短期 MVP

## 2. セルフチェック

- Out of Scope（診断・家族外共有・IoT 連携）の混入なし。
- 粒度: 一次アクターのゴール達成単位（プロフィール登録・編集の完了）で記述し、画面名や単一 API 名で終わっていない。
- 重複: 他 UC（UC-01 認証／UC-02 グループ作成／UC-19 権限管理）と責務を分離。
- 抜け漏れ: 主要アクター（家族メンバー）×主要トリガー（初期セットアップ／迎え入れ／属性変化／是正）を網羅。
- 依存システム・主要 I/O を必須列に明記。
- 未確定事項を「理由／影響／埋め方／宛先／B|NB」形式で記述。
- ID `UC-03` 形式準拠、必須列すべて充足（不明は TBD 明示）。
