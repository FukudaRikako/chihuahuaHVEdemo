# UC-20: 家族メンバーが KPI ダッシュボード（投薬遵守率等）を閲覧する

- 入力: `docs/catalog/use-case-skeleton.md`（UC-20 行）、`docs/business-requirement.md`（§7 To-Be / §9 ロードマップ / KPI 群）
- 採番: `UC-20`（ARD Step 4.1 骨格と一致）
- 対象スコープ: 推奨戦略オプション B（専用 Web アプリ）長期施策「運用 KPI ダッシュボード」のソフトウェア提供シナリオに限定。

## 1. ID / 名称 / 目的（価値）

- **ID**: UC-20
- **名称**: 家族メンバーが KPI ダッシュボード（投薬遵守率等）を閲覧する
- **目的（価値）**:
  - 家族メンバーが投薬・予防薬・週次体重測定などの遵守状況を一目で把握し、抜け漏れ傾向や担当偏りに早期気付く。
  - 通院前サマリ生成や運用改善の意思決定に活用できる定量指標を提供する（家計内コスト最適化と犬の健康寿命最大化への寄与）。

## 2. 主アクター

- **一次アクター**: 家族メンバー（特に主家族メンバーが運用改善目的で参照する想定）。
- **二次アクター / 外部システム**:
  - システム（集計バッチ / 集計サービス）: 投薬・予防薬・体重・サマリ生成等のログを KPI として集計。
  - 通知サービス（参考連携、本 UC の範囲外）: 値の悪化検知時の通知連携余地（本 UC では参照のみ、TBD）。

## 3. 前提条件

- UC-01（サインアップ / ログイン）が完了し、当該家族メンバーが認証済みである。
- UC-02 により家族グループが作成され、当該メンバーが所属している。
- UC-03 で 1 頭以上の犬プロフィールが登録されている。
- 集計対象となる記録（UC-05 投薬、UC-06 予防薬、UC-07 体重、UC-12 通院前サマリ生成 等）が一定期間分蓄積されている（蓄積ゼロでも空状態を明示表示）。
- KPI 定義値（投薬遵守率の算出式・対象期間・分母分子の定義）が確定している。未確定の場合は要確認（TBD）。

## 4. 基本フロー要約

1. 家族メンバーがログイン後、ナビゲーションから「KPI ダッシュボード」を選択する。
2. システムは家族グループに属する犬・家族メンバーの最近の記録を読み出し、所定の KPI を集計する（既定期間: 直近 30 日、TBD: 既定期間値）。
3. システムは以下の KPI カードを表示する:
   - 投薬遵守率（記録ベース）
   - 月次予防薬（フィラリア / ノミダニ）遵守率
   - 週次体重測定実施率
   - 通院前サマリ生成所要時間（直近平均）
   - 家族メンバー別ログイン頻度
4. 家族メンバーは期間フィルタ（直近 7 日 / 30 日 / 90 日）と犬フィルタ（全頭 / 個別犬）を切り替えて再集計させる。
5. 家族メンバーは各 KPI カードから関連する記録一覧（UC-10 タイムライン等）へ遷移し、悪化要因を確認する。

## 5. 主要例外

- **E1: 記録が不足し KPI を算出不能**
  - 条件: 対象期間内に該当ログがゼロ件 → 結果: 「データ不足」状態を明示し、関連記録 UC への導線（UC-05/06/07 等）を提示。算出は行わない（捏造禁止）。
- **E2: KPI 定義未確定の指標**
  - 条件: 算出式が TBD のまま → 結果: 該当 KPI カードを「定義未確定」とラベル表示し、値表示は抑止。
- **E3: 権限不足 / 非家族メンバーアクセス**
  - 条件: UC-19 のアクセス権限管理で参照権が無いユーザー（将来拡張） → 結果: アクセス拒否を表示し操作中止。MVP では家族メンバー全員参照可とする想定（TBD）。

## 6. 主要データ I/O

- **入力（参照）**:
  - 投薬ログ（UC-05）、予防薬ログ（UC-06）、体重ログ（UC-07）
  - 通院前サマリ生成イベント（UC-12）
  - 家族メンバー / ログインイベント（UC-01, UC-02）
  - 犬プロフィール（UC-03）
- **出力（表示専用、永続化なし）**:
  - KPI 集計結果（指標値、対象期間、分母分子）
  - フィルタ条件（期間 / 犬）
- **データ SoR / オーナー**: 各記録 UC のオーナー（投薬ログは UC-05、体重は UC-07 等）。本 UC は **読取専用**。

## 7. KPI

> `docs/recommended-kpi-okr.md` は本リポジトリには未生成のため、`KPI-*` / `OKR-*` ID 付与は省略する（捏造禁止）。下記は `docs/business-requirement.md` §7 To-Be / §9 ロードマップに記載された主要 KPI を本 UC が満たす指標として列挙したもの。

- 投薬遵守率（記録ベース）: 期間内に必要だった投薬回数のうち実施記録があった割合。
- 月次予防薬遵守率: フィラリア / ノミダニ予防薬の期日内投与達成率。
- 週次体重測定実施率: 週 1 回以上の体重測定が記録された週の割合。
- 通院前サマリ生成所要時間: UC-12 における操作開始〜PDF / サマリ出力完了までの所要平均（目標: 30 分 → 5 分）。
- 家族メンバー別ログイン頻度: アクセス偏りの可視化（家族間タスク重複 I6 の運用改善）。
- 重大症状から獣医師連絡までのリードタイム: 取得方式 TBD（記録 UI が必要、要確認）。
