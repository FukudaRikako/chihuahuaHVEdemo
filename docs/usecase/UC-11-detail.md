# UC-11 詳細 — 家族メンバーが体重・食欲の時系列グラフを閲覧する

- 入力骨格: `docs/catalog/use-case-skeleton.md` の UC-11 行
- 一次根拠: `docs/business-requirement.md`（I3 体調変化の検知遅延／§6 To-Be「自動でタイムライン・グラフ更新」／§9 推奨戦略オプション B／ロードマップ中期「体重／食欲グラフ」）
- 対象スコープ: 戦略オプション B（専用 Web アプリ MVP→拡張）配下のソフトウェア提供シナリオに限定。診断行為・閾値判定は対象外。
