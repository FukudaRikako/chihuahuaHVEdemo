# UC-07 詳細 — 家族メンバーが体重測定値を記録する

- 対象事業: チワワファミリーの家庭内ペット健康管理（家族共有型 Web アプリ MVP）
- 関連スケルトン: `docs/catalog/use-case-skeleton.md` UC-07
- 入力: `docs/business-requirement.md`（As-Is 4.1「週次: 体重測定」、To-Be §「体重」中核ログ、Key Issue I3 体調変化検知、Strategic Recommendations オプション B 短期 MVP）
- 戦略整合: オプション B（専用 Web アプリ MVP→拡張）の短期 MVP 中核ログ（体重）。診断行為・家族外共有・スマート体重計等 IoT 自動計測は対象外。

| In範囲 | 体重値（kg）の手入力、測定日時の選択、測定条件メモ、前回値差分の自動表示、家族メンバー間の即時共有、週次測定リマインドとの連動。 |

- Out of Scope の混入なし（診断・IoT 自動計測・EC・家族外共有・体組成を Out 範囲に明記）。
- 粒度: 一次アクター（家族メンバー）のゴール達成単位「体重測定値を記録する」で統一。画面/API 名止まりではない。

- 抜け漏れ: 主要トリガー（週次測定／追加測定／今日のタスク導線）をカバー。
- 依存・I/O: 認証・犬プロフィール・タイムライン・グラフ・通院前サマリ・検索・通知を明記。
- 未確定事項: 3 件を理由/影響/埋め方/宛先/B|NB 形式で記載。
