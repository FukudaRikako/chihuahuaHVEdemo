# UC-08 詳細 — 家族メンバーが食事・排泄・体調観察を記録する

- 対象事業: チワワファミリーの家庭内ペット健康管理（家族共有型 Web アプリ MVP）
- 関連スケルトン: `docs/catalog/use-case-skeleton.md` UC-08
- 入力: `docs/business-requirement.md`（As-Is 4.1〜4.2、To-Be 日次記録、Strategic Recommendations オプション B）
- 戦略整合: オプション B（専用 Web アプリ MVP→拡張）の「日次記録」中核機能。診断行為・家族外共有・IoT 連携は対象外。

| データSoR/オーナー部門 | SoR: 本 Web アプリの ObservationRecord ストア。オーナー: 主家族メンバー（家族グループ単位）。 |

## セルフチェック

- Out of Scope の混入なし（診断・写真・IoT・家族外共有を Out 範囲に明記）。
- 粒度: 一次アクター（家族メンバー）のゴール達成単位「食事・排泄・体調観察を記録する」で統一。画面/API 名止まりではない。
- 重複なし: 投薬は UC-05、予防薬は UC-06、体重は UC-07、症状＋写真は UC-09 と明確に分離。
- 抜け漏れ: 主要トリガー（日次給餌/排泄/観察直後）をカバー。
- 依存・I/O: 認証・犬プロフィール・タイムライン・グラフ・通院前サマリ・検索を明記。
- 未確定事項: 3 件を理由/影響/埋め方/宛先/B|NB 形式で記載。
