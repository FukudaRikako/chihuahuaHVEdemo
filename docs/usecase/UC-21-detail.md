# UC-21 詳細: 家族メンバーが過去の記録を犬・期間・カテゴリで検索する

- 対象事業: チワワファミリーの家庭内ペット健康管理（家族共有型 Web アプリ）
- 推奨戦略: オプション B（専用 Web アプリ MVP→拡張）
- 関連課題: I7（写真・症状メモの分散／検索に時間）
- 入力骨格: `docs/catalog/use-case-skeleton.md` UC-21

## ユースケース詳細

- **ID**: UC-21
- **名称**: 家族メンバーが過去の記録を犬・期間・カテゴリで検索する
- **業務エリア or ジャーニー段階**: 記録活用（情報参照・通院前準備支援）
- **目的**: 蓄積された投薬・体重・症状メモ・写真等の記録から、目的の情報（例: 「ちょこの直近3か月の咳の記録」「ルナの先月の食事メモ」）を短時間で発見し、家族間の状況共有や通院前情報整理（UC-12）の精度を高める。
- **提言/論点ID**: オプション B（推奨戦略）／関連課題 I7
- **一次アクター**: 家族メンバー
- **二次アクター/外部システム**: なし（本ソリューション内で完結）
- **トリガー**: 家族メンバーが過去の記録を参照したい状況（通院前情報の補強、症状の再発確認、家族間で「いつのことだっけ？」の確認 等）でアプリの検索画面を開く。
- **前提条件**:
  - 家族メンバーが認証済みであり、家族グループに所属している（UC-01／UC-02）。
  - 検索対象となる記録（投薬・体重・症状・写真等）が UC-05〜UC-09 等で蓄積されている。
- **基本フロー要約**:
  1. 家族メンバーが検索画面を開き、対象犬（まめ／こむぎ／ちょこ／ルナ／全頭）、期間（プリセットまたは任意範囲）、カテゴリ（投薬／予防薬／体重／食事／排泄／症状・写真）を指定する。
  2. 任意でフリーキーワード（症状メモ本文に対する部分一致）を入力する。
  3. システムが家族グループ内の記録から条件に合致するものを時系列降順で一覧表示する（写真はサムネイル併記）。
  4. 家族メンバーが一覧から記録を選択し、詳細ビュー（統合タイムライン UC-10）へ遷移する。
- **主要代替/例外**:
  - 該当 0 件: 「条件に合致する記録がありません」を表示し、期間拡張・カテゴリ解除を提案する。
  - 大量ヒット（しきい値超過）: 上位 N 件をページング表示し、絞り込み追加を案内する。
  - 権限外データ参照: 他家族グループの記録は検索結果に含めない（UC-19 のアクセス制御に従う）。
- **事後条件**: 検索条件と結果一覧が画面上に保持され、詳細ビューや通院前サマリ生成（UC-12）への動線が利用可能。記録自体は変更されない（参照系ユースケース）。
- **主要データI/O**:
  - 入力: 検索条件（対象犬 ID／期間／カテゴリ／キーワード）
  - 出力: 記録一覧（記録 ID／犬／日時／カテゴリ／要約／写真サムネイル）
  - 参照対象データオブジェクト: 投薬記録／予防薬記録／体重記録／食事・排泄・観察記録／症状メモ／添付写真
- **データSoR/オーナー部門**: 本ソリューション（家族共有型 Web アプリ）／本ファミリー（家族グループ単位で所有）
- **依存システム**: 本ソリューション内のイベントストア（ペット軸時系列）および写真ストレージ
- **In範囲**: 家族グループ内の蓄積記録に対する条件指定検索・一覧表示・詳細遷移
- **Out範囲**: 症状の医療診断・原因推定／家族外（獣医師等）への直接共有／OCR や音声検索などの高度入力／外部 SaaS への記録エクスポート
- **Out理由（短文）**: 推奨戦略オプション B のスコープ外（診断機能・家族外共有・高度入力は MVP〜拡張範囲外）。
- **KPI紐づけ**: TBD（`docs/recommended-kpi-okr.md` 未生成のため未確定。候補としては「検索結果到達までの平均所要時間」「通院前サマリ作成時の参照ヒット率」が想定されるが要確認）
- **優先度**: P2
- **未確定事項**:
  - TBD: キーワード検索の対象範囲（症状メモ本文のみか写真キャプション等も含むか）／影響: 検索ヒット率・実装コスト／埋め方: プロダクトオーナーへの確認／宛先: 主家族メンバー／NB（Non-Blocker：MVP では症状メモ本文のみで合意可能）
  - TBD: 写真のタグ付け要否（I7 で「タグ付け不在」が原因と指摘）／影響: 検索精度／埋め方: UC-09（症状・写真記録）との設計協議／宛先: 設計担当／NB
  - TBD: 検索結果のエクスポート要否（CSV/PDF）／影響: UC-18 との重複可否／埋め方: ユーザーヒアリング／宛先: 主家族メンバー／NB
