# 4匹のチワワ健康管理 seed Q&A

| ID | 質問 | 回答 |
|---|---|---|
| Q1 | 初回リリースで管理するペットは何匹か | まめ、こむぎ、ちょこ、ルナの4匹を対象にする |
| Q2 | 医療診断機能は必要か | 不要。診断は行わず、記録、リマインド、通院時の情報整理に限定する |
| Q3 | 家族共有では何を確認したいか | 食事、投薬、予防薬、通院、体重測定が対応済みかを確認したい |
| Q4 | 通院前に出したい情報は何か | 直近の体重、食欲、便の状態、症状メモ、写真、投薬履歴を犬ごとにまとめたい |
| Q5 | リマインド対象は何か | 毎日の投薬、月次の予防薬、ワクチン、通院予約、週次の体重測定を対象にする |
| Q6 | 非機能で重視するものは何か | スマホでの入力しやすさ、家族だけが見られるアクセス制御、写真保存、通知の分かりやすさを重視する |
